# Employee Management System

## Project Overview
Employee Management System is a web-based application developed using Java, JSP, Servlets, JDBC, MySQL, Maven, and Apache Tomcat.

The system allows administrators to manage employee records and employees to access their profiles through a secure login system.

This project was originally developed as an Advanced Java college project and later enhanced with additional features.

## Technologies Used
- Java
- JSP
- Servlets
- JDBC
- MySQL
- Maven
- Apache Tomcat
- HTML
- CSS
- JavaScript
- Brevo Email API
- Render.com
- Aiven MySQL

## Features

### Authentication & Security
- Role-based login (Admin / Employee)
- SHA-256 password hashing
- First-time login password change enforcement
- Session management and secure authentication

### Employee Features
- View profile information
- Change password
- Upload profile picture
- Remove profile picture
- Dark mode support

### Admin Features
- Add employee
- View employee records
- Update employee information
- Delete employee records
- Employee search and management
- Role-based dashboard

### Announcement System
- Send announcements to:
  - All employees
  - Same department
  - Custom employee groups
  - Selected employee ranges

### Email Notifications
- New employee account creation email
- Temporary password email
- Employee update notification email
- Employee deletion notification email

### Deployment
- Converted to Maven project
- Packaged as WAR file
- Deployed using Apache Tomcat
- Database hosted on Aiven
- Application deployed on Render
